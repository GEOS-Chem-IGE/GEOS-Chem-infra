GEOS-Chem-Infra
===============

This repository contains software infrastructure to run the [GEOS-Chem](https://geoschem.github.io/index.html) global atmospheric chemistry model at the [IGE lab](https://www.ige-grenoble.fr/?lang=en).


Contents
--------

* [doc](doc/): documentation and tutorials for installation and running GEOS-Chem on [dahu](https://github.com/GEOS-Chem-IGE/GEOS-Chem-infra/blob/issue14/update-readme/doc/dahu-ciment.md) and [ige-calcul] servers.
